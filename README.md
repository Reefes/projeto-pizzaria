Site de Pizzaria (Projeto de Estudo)

Este é um projeto focado no estudo prático de desenvolvimento web (HTML e CSS).

Como foi feito:
A estrutura inicial e o boilerplate do código foram gerados com o auxílio de Inteligência Artificial. Meu principal foco neste projeto foi fazer a "engenharia reversa": analisar o código gerado, modificar a estrutura, aplicar estilos personalizados e entender a fundo a lógica por trás da interface.

Objetivo:
Desenvolver a capacidade de ler, compreender e refatorar códigos de terceiros, uma habilidade essencial no dia a dia do desenvolvimento.
